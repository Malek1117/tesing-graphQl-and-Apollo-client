# SIMPLE Login app using GraphQL and Apollp Client

Clone the project

```bash
  git clone https://github.com/Malek1117/tesing-graphQl-and-Apollo-client.git
```

Go to the project directory


Start the client
```bash
  cd client
  npm install
  npm start 
```

Start the server
```bash
  cd server
  npm install
  npm run server
```
